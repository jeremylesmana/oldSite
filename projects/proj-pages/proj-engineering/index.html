ENGINEERING DESIGN TECHNOLOGY

          This class taught me the basic fundamentals of the software Solidworks and Sketchup. Using these software, I've learned to efficiently model three-dimensional objects with as little steps as possible. Not only that we achieved to model these objects on the computers, but we were able to transmit our creations into real-life models; such as the examples listed in this section.

          Aside from learning on the computer, we also learned how to draw and sketch with our hands. We learned how to draw isometric and orthographic drawings on paper. We also learned how to dimension objects properly using dimension features. We also performed many experiments with our real-life models; which resulted in class competitions.

COMPUTER AIDED DESIGN

          This class taught me the basics of constructing different parts by using a variety of tools. I had the opportunity to handle various tools such as saws and drills, and to work with different materials such as plywood, wooden planks, metal rods, metal plates, etc... Most importantly, this class taught me the different safety features of the tools I used to prevent future injuries or accidents. Aside from working with real life constructing projects, we also dug deeper into the Solidworks software which allows us to create more parts using the new features that we learned.
